### 学习任务

#### 掌握搜索 

##### global search 全局搜索

- `shift + ctrl + f`

##### go to Symbol in Workspace 在工作区搜索

- `ctrl + t`

##### go to Symbol in File 在当前文件搜索

- `ctrl + shift + o`
- `:` 按照类别分组

##### search commands 全局命令搜索

- `ctrl + shift + p`

##### quick file navigation 搜索文件

- `ctrl + p`
- `>` 回到全局命令搜索
- `@` 在当前文件搜索
- `#` 全局搜索

##### switch files 在打开文件之间切换

- `ctrl + tab`
